# CTRLBuild Project

This is a modern web application built with React, Vite, and Three.js. It features interactive WebGL particle text systems, fluid cursor effects, and high-performance animations using Lenis scroll.

## Tech Stack
- React 19
- Vite
- Three.js
- Lenis (Smooth Scrolling)
- Lucide React (Icons)
- Typewriter Effect

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
